내 컴퓨터에서 깃헙으로 업로드하는 방법
처음 업로드 할 때


1. 깃헙 셋팅
  1. git config --global user.name "your_name"
  2. git config --global user.email "your_email"
2. 깃헙 셋팅 확인
  1. git config --list
3. 깃헙 초기화 및 업로드
  1. git init
  2. git add .
    .(점)은 모든 파일을 뜻한다.
  3. git status
  4. git commit -m "your_message"
  5. git remote add origin your_repository_address
  6. git remote -v
  7. git push origin master
다시 업로드 할 때


1. 깃헙 업로드
  1. git add .
  2. git commit -m "your_message"
  3. git push origin master
