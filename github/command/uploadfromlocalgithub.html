How to upload from local
When you upload for the first time
1. Setting GitHub
git config --global user.name "your_name"
git config --global user.email "your_email"
2. Check
git config --list
3. Init GitHub
git init
git add .
.(dot) means all files
git status
git commit -m "your_message"
git remote add origin your_repository_address
git remote -v
git push origin master
if you don't know visit this site
