깃헙에 호스팅 하는 방법
  1. Make a new Repository(새로운 리포지토리 만들기)
  2. (Repository = 코드 저장소)
  3. Settings(Toolbar) -> Pages(Sidebar)
  4. Change "None" to "Branch" or "Main" and Click "Save"
    (if you choose "Branch", your webpage will be changed whem you upload your HTML file from local. if you choose "Main", you need to upload your HTML file on GitHub site.
  5. Click Actions(Toolbar) Refresh if there's nothing
    (it takes time to host webpage on GitHub)
    (it will build, report-build-status, deploy)
  6. The address of your webpage is in Setiings -> Pages
